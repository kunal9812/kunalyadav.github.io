# kunalyadav.github.io

Hey! I'm Kunal Yadav, a CS student and developer passionate about systems programming, open-source, and building things that matter.

This is my personal portfolio website — built in 3D. Check it out live at [kunalyadav.github.io](https://kunal9812.github.io/kunalyadav.github.io/)

## Tech Stack

- Next.js 15
- React 19
- React Three Fiber (Three.js)
- Drei
- GSAP
- Zustand
- Tailwind CSS
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.

## Project Structure

```
app/
├── components/     # Hero, Experience, Footer, 3D models
├── constants/      # Projects, work timeline, footer links (edit these!)
├── stores/         # Zustand state (theme, scroll, portal)
└── types/          # TypeScript interfaces
public/
├── models/         # 3D .glb model files
├── icons/          # SVG icons for footer
└── fonts           # Custom fonts (soria, vercetti)
```
